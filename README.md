# PiggyWeb - 小猪春节日记

## 项目简介

一款温馨的互动养成游戏，讲述一只在春运中走失的小猪与玩家共度春节的故事。

## 游戏概述

### 故事背景

小猪在春运期间意外走失，与玩家相遇并一起度过春节。玩家需要照顾小猪，在准备过年的过程中触发各种有趣的故事事件。最终在跨年夜，小猪的主人小科会前来寻找，故事迎来温馨结局。

### 核心玩法

#### 1. 故事收集系统

- **触发机制**：时间推进自动出现对话框，或通过点击对话框/戳小猪主动触发
- **收集图鉴**：记录所有已解锁的故事事件（如：包饺子时小猪掉进面粉里）
- **日记形式**：以养宠物日记的方式呈现，每天包含多个故事事件

#### 2. 宠物照顾系统

- **喂食功能**：通过喂食来照顾小猪（具体食物类型待定）
- **互动反馈**：照顾行为影响小猪的状态和故事触发

#### 3. 戳戳乐互动

- **触摸交互**：可以戳小猪进行互动
- **即时反馈**：戳击触发动画和音效反馈

## 游戏机制

### 时间系统

- **进度机制**：完成当天所有剧情后才能进入下一天
- **结局触发**：春节当天打开游戏自动触发结局剧情（小科登场）
- **补完机制**：未收集的剧情可通过小科的信件方式补回

### 收集平衡

- 收集压力适中，既不会过于困难，也不会过于简单
- 鼓励玩家每天体验，逐步解锁故事
